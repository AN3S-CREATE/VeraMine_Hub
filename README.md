# VeraMine Hub

This project is a React-based dashboard for VeraMine Hub.

## Development

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run development server:
    ```bash
    npm run dev
    ```

## Deployment

This project is configured to deploy to GitHub Pages automatically via GitHub Actions.

When you push to the `main` or `master` branch, the deployment workflow will run.

### Manual Setup
If the deployment doesn't happen automatically, go to **Settings > Pages** in your GitHub repository and ensure that **Source** is set to "GitHub Actions".
